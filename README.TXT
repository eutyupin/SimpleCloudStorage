
   На данный момент проект умеет регистрировать пользователей, авторизовать, копировать файлы от клиента к серверу
   и от сервера к клиенту (только файлы по одному, не папки) удалять папки/файлы на стороне хранилища и создавать пустые папки.
   На клиентской части имеется интерфейс.

   Для запуска проекта нужен JDK 11.0.2 Или выше.
   Проект построен на maven.

1. Запускаете IntelliJIdea
2. В окне выбора проекта выбираете "Get from VCS"
3. В предложенное вам поле вставляете ссылку https://github.com/eutyupin/SimpleCloudStorage
4. Нажимаете кнопку "Clone"
5. После того как проект склонируется и загрузятся зависимости maven
   на панеле maven нажимаете кнопку "Reload All maven Projects" затем "Execute Maven Goal"
   и в строке введите mvn install
6. После компиляции файлов нужно перейти в папку проекта и запустить сервер и клиента(ов)

   Для запуска сервера перейти в папку:
   C:\Users\%ИМЯ ВАШЕГО ПОЛЬЗОВАТЕЛЯ%\IdeaProjects\SimpleCloudStorage\server\target\
   зажать SHIFT и нажать правую кнопку мыши, далее выбрать пункт контекстного меню "Открыть окно PowerShell сдесь" (Windows10)
   либо в консоли перейти к этой папке и ввести следующую комманду:
   java -jar server-1.081-spring-boot.jar НОМЕР_ПОРТА (если не ввести номер порта, то порт будет назначен по умолчанию 8189)

   Для запуска клиента(ов) перейти в папку:
   C:\Users\%ИМЯ ВАШЕГО ПОЛЬЗОВАТЕЛЯ%\IdeaProjects\SimpleCloudStorage\client\target
   зажать SHIFT и нажать правую кнопку мыши, далее выбрать пункт контекстного меню "Открыть окно PowerShell сдесь" (Windows10)
   либо в консоли перейти к этой папке и ввести следующую комманду:
   java -jar client-1.081-spring-boot.jar ИМЯ_ХОСТА НОМЕР_ПОРТА (если не ввести номер порта и имя хоста,
                        то они будут назначен по умолчанию хост localhost порт 8189)
   Также если вам не нужно менять порт и хост (если например запускаете локально на одной машине), клиента можно запустить просто двойным щелчком мыши

8. Если вы хотите запустить проект в IDE, то для запуска сервера необходимо:

   Запустить на выполнение класс server/src/main/java/ru/simplecloudstorage/ServerApp.java (как обычное приложение Java SHIFT+CTRL+F10)

   Для запуска клиента(ов) необходимо на панеле maven выбрать следующее:
   нажать кнопку "Reload All maven Projects"
   далее выбрать пункты client -> Plugins -> javafx -> javafx:run

   При старте приложения необходимо зарегистрироваться и авторизоваться. На серверной части для каждого пользователя
   создается своя папка с именем пользователя. И только в нее текущий пользователь имеет доступ.
